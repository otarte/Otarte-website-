# OTARTE · Web lista para GitHub Pages

## Publicar (sin instalar nada)
1) Crea un repositorio en GitHub (ej: `otarte-web`).
2) Sube TODO el contenido de esta carpeta (debe verse `index.html` en la raíz).
3) Activa: Settings → Pages → Deploy from a branch → `main` / `/(root)`.

## Dominio otarte.es (cuando toque)
En Settings → Pages → Custom domain: `otarte.es`
Luego en tu proveedor del dominio, pon los registros A de GitHub Pages.

## Notas
- `.nojekyll` evita que GitHub toque carpetas/archivos.
- `sitemap.xml` y `robots.txt` ya están incluidos.
